ISA
Why this dashboard

A shared instrument
for a shared mission

The ISA's mandate spans science, law, environment, and equity — across 169 member states, 31 contractors, and a global public. This dashboard gives every actor a single, trusted view of collective progress.

01
Guide daily work
ISA staff and the Secretary-General see real-time progress across all four mandate pillars — so decisions are grounded in current data, not last year's reports.
02
Align stakeholders
Member states, contractors, and civil society all look at the same dashboard — building a shared understanding of where things stand and what comes next.
03
Demonstrate leadership
Proactive transparency builds institutional trust. Publishing this data sends a clear signal: the ISA is confident, forward-looking, and accountable to humankind.

Implementation

Getting to live in
three phases

A realistic build sequence — starting with what's already open and available, expanding as data pipelines mature.

Phase 1 · Months 1–3
Foundation
Live biodiversity + static context
Connect OBIS ISA node live API
Display all 31 contracts with mineral type and zone
Publish regulatory milestones timeline (static)
Interactive map: contract zones + APEI boundaries
Host publicly at isa.org.jm/dashboard
Phase 2 · Months 4–8
Depth
Contractor data + science tracking
Contractor environmental report submission tracker
Mining code article-by-article status tracker
GBIF integration for species-level drill-down
Research program registry per zone
Multi-language support (EN, FR, ES, ZH)
Phase 3 · Month 9+
Full mandate
Shared benefit + live feeds
Common Heritage Fund disbursement tracker
EPC activity and Economic Planning Commission log
Live session updates during ISA Council meetings
API for civil society and researchers to build on top
Downloadable open data exports per pillar
